# Fixed Broadband Deployment Data

### Setting up locally
1. Install node modules.  
   ```
   npm install
   ```

2. Build the docs (HTML, LESS/CSS, JS).  
   
   Development
   ```
   grunt
   ```
   Production
   ```
   grunt build:release
   ```
   
3. Run the app using a local server.  
   ```
   localhost:8080
   ```
